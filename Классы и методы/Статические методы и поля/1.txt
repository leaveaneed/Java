Вызов через имя класса
A.printVars();
Статический метод printVars() принадлежит классу A. Поэтому мы можем вызвать его, используя имя класса A.

Вызов через объект класса
A obj = new A();
obj.printVars();
Создается объект класса A с именем obj.
Статический метод printVars() вызывается через объект.
Однако статический метод не привязан к конкретному объекту и на самом деле вызывается на уровне класса, поэтому этот способ может сбивать с толку.

Вызов через наследника (если класс A наследуется)
public class B extends A {
    // Класс B наследует статический метод printVars() из A
}
B.printVars(); // Вызов через имя класса-наследника
Класс B наследует метод printVars() от A.
Статические методы не переопределяются, но могут быть унаследованы и вызваны через имя класса-наследника.

Вызов через объект наследника
B obj = new B();
obj.printVars();
Хотя вызов идет через объект, метод printVars() остается статическим и принадлежит классу A.
