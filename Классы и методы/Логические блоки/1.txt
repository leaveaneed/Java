Вывод программы:
static logic
logic (1) id= 0
logic (2) id= 1
ctor id= 100

Объяснение:
"static logic" выводится первым, потому что статический блок выполняется один раз при первой загрузке класса.
"logic (1) id= 0" выводится следующим, потому что объект ещё не полностью инициализирован, и на момент выполнения этого блока поле id имеет значение по умолчанию — 0.
"logic (2) id= 1" выводится после того, как поле id было инициализировано значением 1 (до выполнения конструктора).
"ctor id= 100" выводится в самом конце, когда конструктор присваивает новое значение полю id — 100.
