При объявлении:

public final int a = 10;

В конструкторе:

public A(int value) {
    this.a = value;
}

В нестатическом блоке инициализации:
 
{
    a = 20;
}

В статическом блоке (для static final):
 
static {
    a = 30;
}

Важно:
Поле final обязательно должно быть проинициализировано до завершения конструктора.
Если это static final, то инициализация должна произойти в статическом блоке или при объявлении.
